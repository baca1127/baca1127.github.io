# 兒童遊戲集

三個適合不同年齡的 HTML 網頁遊戲，全部使用繁體中文。

---

## 🫧 戳泡泡（`index.html`）

| 項目 | 內容 |
|------|------|
| 適合年齡 | 3 歲以上 |
| 玩法 | 在 30 秒內點擊戳破彩色泡泡，連擊可獲得額外分數 |
| 操作 | 滑鼠點擊 / 手指觸碰 |
| 計分 | 每戳一顆 +1 分，連擊 3 次以上每顆再 +1；漏掉 3 顆遊戲結束 |

---

## 🐾 動物找找看（`animals.html`）

| 項目 | 內容 |
|------|------|
| 適合年齡 | 4~5 歲（幼兒園） |
| 玩法 | 畫面顯示 6 張動物卡，找出指定的動物（如「🐶 小狗」） |
| 操作 | 點擊動物卡片 |
| 計分 | 答對 +10 分，答錯 -2 分（最低 0 分） |
| 特色 | 無時間壓力、答對有彩色紙花特效、鼓勵與安慰訊息 |

---

## 🚀 數學小英雄（`math.html`）

| 項目 | 內容 |
|------|------|
| 適合年齡 | 6~7 歲（小學一年級） |
| 玩法 | 加減法選擇題，4 選 1 |
| 關卡 | 共 3 關，每關 10 題 |
| 內容 | 第 1 關：0~10 加減（附圓點數數輔助） |
|      | 第 2 關：0~20 加減 |
|      | 第 3 關：0~50 加減 |
| 計分 | 答對 +10 分，答錯 -3 分 |
| 星等 | 通關後依答對數給予 1~3 顆星 |

---

## 遊玩方式

直接用瀏覽器開啟對應的 `.html` 檔案即可，無需安裝任何軟體或網路連線。
